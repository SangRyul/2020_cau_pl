## Summary

## Prerequisites

- python >= 3.7
- numpy == 1.19.4

## Module Install

```sh
pip install numpy
```
## Execute
```sh
python interpreter.py
# 그다음 선택지에 따라 1 , 2 또는 3입력 (1: file 모드, 2: repl 모드 3: quit
```
